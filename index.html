Deeply Engineered

Each digit is built from precision-cut transparent layers. Only one layer is illuminated at a time, creating a floating number with real physical depth.

  1. 01

    Acrylic Layer Stack

    Multiple optical-grade acrylic plates arranged with exact spacing.

  2. 02

    Precision Engraving

    Each layer contains a finely engraved digit for controlled light diffusion.

  3. 03

    Single Layer Illumination

    A focused LED activates only one layer at a time — making the digit appear suspended in space.

  4. 04

    Controlled Light Path

    Light travels through the acrylic and scatters only at engraved regions, ensuring clarity and contrast.
